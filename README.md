# JavaScript Projects Repository

This repository contains a collection of simple web-based projects built with HTML, CSS, and JavaScript. Each project is designed to demonstrate a different front-end concept and can be opened directly in a browser.

## Projects Included

### 1. Weather App
Location: grp3-weather-app

This project is a basic weather application that allows users to view weather information for a selected location. It uses JavaScript to fetch and display weather data in a user-friendly interface.

### 2. Resume Template
Location: lab2-resume-template

This project is an interactive resume template that presents personal information, skills, work experience, projects, education, and achievements. It also includes a search feature that helps users quickly find keywords inside the resume.

### 3. Quiz Application
Location: lab3-quiz

This project is a simple multiple-choice quiz app. Users can select an answer for each question, receive points for correct answers, and view their final score and percentage at the end.

## Getting Started

1. Clone or download this repository to your local machine.
2. Open the folder for the project you want to view.
3. Open the HTML file in a web browser.

Example:
- Open grp3-weather-app/index.html for the weather app
- Open lab2-resume-template/resume/resume.html for the resume template
- Open lab3-quiz/quiz.html for the quiz app

## Requirements

- A modern web browser such as Chrome, Edge, or Firefox
- Optional: a live server extension like Live Server in VS Code for a smoother local preview

## Notes

These projects are beginner-friendly examples intended for learning and practice. They can be improved further with better styling, validation, additional features, and more interactive behavior.
